## 1. 프로젝트 계획 이유
[**MyTodoList**](https://lee-sang-beom.github.io/My-Todo-List/) : 기본적인 HTML, CSS, JS의 사용법을 익히고, 직접 사용해보며, 앞으로의 개발 공부 방향을 정립하기 위해 해당 프로젝트를 계획하였습니다.

### - 제공기능
	1. 간단한 로그인 기능
	2. TodoList 추가/삭제
	3. 지역, 날씨, 온도, 습도 정보 제공
    4. 시간 정보 제공
    5. 유명한 문장 출력 기능
